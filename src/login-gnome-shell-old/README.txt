COMPILE GNOME-SHELL LIBRARY

	- In the project folder run command: glib-compile-resource gnome-shell-theme.gresource.xml. So the project folder is united_purple or united_purple_solid
	- That will generate file named: gnome-shell-theme.gresource
	- This file is the one that need to be placed to /usr/share/gnome-shell/ folder. Always remember backup of original.
